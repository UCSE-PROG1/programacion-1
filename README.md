## Herramientas necesarias

Para trabajar con el material de este repositorio necesitás tener instalado:

### .NET 8 SDK
El entorno de ejecución y compilación de C#. Descargalo desde el sitio oficial:
- [https://dotnet.microsoft.com/download/dotnet/8.0](https://dotnet.microsoft.com/download/dotnet/8.0)

Para verificar la instalación ejecutá en la terminal:
```bash
dotnet --version
```

### Visual Studio Code
Editor de código recomendado para la materia. Liviano, gratuito y con excelente soporte para C#.
- [https://code.visualstudio.com/](https://code.visualstudio.com/)

Extensiones recomendadas:
- **C# Dev Kit** (Microsoft)
- **GitLens**
- **NuGet Package Manager GUI**

### Git
Sistema de control de versiones necesario para clonar el repositorio y trabajar con las entregas.
- [https://git-scm.com/](https://git-scm.com/)

Para verificar la instalación:
```bash
git --version
```

## Cómo usar este repositorio

1. **Cloná el repositorio** en tu máquina local:
   ```bash
   git clone <url-del-repositorio>
   cd programacion-1
   ```

2. **Navegá a la unidad** que te corresponde según el cronograma de la materia. Cada carpeta tiene su propio `README.md` con la teoría y ejemplos.

3. **Leé la teoría** en el `README.md` de cada unidad antes de arrancar con los ejercicios.

4. **Realizá los ejercicios** que se encuentran dentro de la subcarpeta de ejercicios de cada unidad (por ejemplo, `001-introduccion-poo-netcore-csharp/001-ejercicios-net/`).

5. **Consultá las dudas** con tu docente o en el espacio de comunicación de la materia.
